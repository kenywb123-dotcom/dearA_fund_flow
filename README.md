# A股板块主力资金流向工具

基于东方财富数据的 A 股板块主力资金流向可视化工具，支持 **概念板块** 和 **行业板块** 双模式，自动绘制分时折线图并生成资金汇总表。

## 功能特点

- 概念板块 / 行业板块 双模式切换
- 主力资金净流入 / 流出 TOP N 排行（N 可自定义）
- 白名单：指定板块强制显示在图中
- 黑名单：过滤掉不需要的板块，不参与排名
- 自动绘制分时折线图 + 右侧资金汇总表
- 图表自动保存并打开
- GUI 图形界面，无需命令行操作

## 快速开始

### 方式一：直接运行 exe

下载 `dist/` 文件夹，双击 `A股板块资金流向.exe`

首次运行会自动创建黑白名单模板文件，填入板块名称即可使用。

### 方式二：运行源码

```bash
pip install matplotlib
python fund_flow_app.py
```

也可单独运行概念版或行业版：

```bash
python sector_fund_flow.py          # 概念版
python industry_fund_flow.py        # 行业版
```

## 黑白名单配置

打开对应的 txt 文件，每行一个板块名称，`#` 开头的行会被忽略：

| 文件 | 说明 |
|------|------|
| `概念板块白名单.txt` | 概念板块强制显示 |
| `概念板块黑名单.txt` | 概念板块过滤排除 |
| `行业板块白名单.txt` | 行业板块强制显示 |
| `行业板块黑名单.txt` | 行业板块过滤排除 |

首次运行或文件不存在时会自动创建空白模板。

## 打包为 exe

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name "A股板块资金流向" fund_flow_app.py
```

## 注意事项

- **运行时间**：建议在交易日 09:30-15:00 运行，非交易时段数据不更新
- **反爬限制**：东方财富有反爬机制，短时间内多次运行可能会被临时封 IP
- **被封处理**：如遇到数据获取失败，请等 3-5 分钟再试，或切换网络/代理节点
- **数据来源**：东方财富（data.eastmoney.com / push2.eastmoney.com）
